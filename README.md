# SCOUT Frontend

Local-first resume and profile management system - Frontend application.

## Overview

Next.js 14 application with App Router, Tailwind CSS, and shadcn/ui components for building accessible, privacy-focused user interfaces.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Node.js 20

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy environment variables:
   ```bash
   cp .env.example .env.local
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run format` - Run Prettier
